# Pixel Jump

Jogo de plataforma HTML5 feito com:
- HTML
- CSS
- JavaScript

## Controles
- A/D ou Setas = mover
- Espaço/W/Seta pra cima = pular

## Como rodar
Abra o arquivo index.html no navegador.

## GitHub Pages
Envie os arquivos para um repositório e ative:
Settings > Pages > Deploy from branch
